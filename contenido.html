QUERY() Avanzado
Concepto Clave
QUERY() es el motor SQL nativo de Google Sheets. Con él construyes reportes dinámicos, tablas cruzadas y análisis multicritrio sin salir de tu hoja — y sin escribir una sola macro.

Cláusulas Core: SELECT · WHERE · ORDER BY · LABEL

SELECT con formato
Controla exactamente qué columnas retornar y en qué orden. Combina con LABEL para renombrar cabeceras y FORMAT para aplicar máscaras de fecha o número directamente en la consulta.
WHERE con condiciones encadenadas
Filtra filas con operadores AND, OR y NOT. Aprende a filtrar por contiene, empieza con, rango de fechas y valores dinámicos usando referencias de celda dentro de la cadena de consulta.
ORDER BY · LIMIT · OFFSET
Ordena resultados ascendente o descendentemente por múltiples columnas. Usa LIMIT para muestras y OFFSET para paginación de datasets grandes.
// Ejemplo: top 10 ventas de México en 2025, ordenadas de mayor a menor QUERY(datos, "SELECT B, SUM(C) WHERE D='MX' AND YEAR(A)=2025 GROUP BY B ORDER BY SUM(C) DESC LIMIT 10 LABEL SUM(C) 'Total'")

GROUP BY y Funciones de Agregación

SUM · AVG · COUNT · MAX · MIN
Consolida datos por categorías usando las cinco funciones de agregación de QUERY. Aprende cuándo usar COUNT vs COUNT(columna) y cómo combinar varias agregaciones en un solo SELECT.
Agrupamiento multinivel
Agrupa por más de una columna simultáneamente para crear jerarquías de análisis. Ejemplo: consolidar ventas por región, luego por categoría de producto y finalmente por mes.

PIVOT Dinámico: Tablas Cruzadas sin Fórmula Manual

Cláusula PIVOT
Transforma valores únicos de una columna en cabeceras de columna nuevas de forma automática. Cuando lleguen nuevas categorías al dataset, la tabla se actualiza sola — sin necesidad de ajustar nada.
GROUP BY + PIVOT combinados
La combinación más poderosa de QUERY: agrupa filas Y pivotea columnas en una sola consulta, generando una tabla dinámica completa con una sola fórmula.

Filtros por Fecha, Texto y Condiciones Múltiples

Operadores de fecha en QUERY
Filtra con YEAR(), MONTH(), DAY() y rangos con date 'YYYY-MM-DD'. Construye filtros dinámicos que se ajustan al mes actual usando concatenación de texto en la consulta.
Filtros de texto: contains · starts with · matches
Usa contains para búsqueda parcial, starts with para prefijos y matches con expresiones regulares para patrones complejos. Mucho más potente que BUSCARV para datos no exactos.

QUERY Anidado y Combinación con ARRAYFORMULA

Subconsultas: QUERY dentro de QUERY
Usa el resultado de una consulta como fuente de otra para filtrar sobre agregados — algo que no es posible con WHERE normal. Patrón clave para arquitecturas de múltiples hojas.
QUERY + ARRAYFORMULA
Extiende columnas calculadas dentro de QUERY usando ARRAYFORMULA como fuente de datos. Crea columnas derivadas en tiempo real sin alterar la hoja de origen.
QUERY + IMPORTRANGE
Consulta datos de otras hojas de cálculo externas directamente con QUERY(IMPORTRANGE(...), "..."). Arquitectura de datos distribuidos sin necesidad de copiar datos manualmente.

Arquitectura: Hoja de Datos vs. Hoja de Reporte

Separación de responsabilidades
El patrón profesional: una hoja recibe datos crudos (nunca se toca), otra hoja contiene solo QUERYs que generan los reportes. Si cambian los datos, los reportes se actualizan solos.
Dashboard con QUERY como motor
Construye un tablero de control completo donde cada sección es una QUERY diferente apuntando a la misma fuente. Escalable, mantenible y actualizable con un clic.

Scripts de Apps Script generados con IA
Concepto Clave
No necesitas aprender a programar. ChatGPT, Claude y Gemini escriben el código de Apps Script por ti — tú solo necesitas saber qué pedirle, cómo pegarlo y cómo hacerlo funcionar. En esta sesión usamos IA como copiloto para construir automatizaciones reales desde cero.

El Flujo de Trabajo: IA → Editor → Sheets

Entender el editor sin ser programador
Conoce el entorno mínimo que necesitas: dónde pegar el código que genera la IA, cómo ejecutarlo, cómo leer el registro de errores y cómo darle permisos a tu script. Todo en menos de 10 minutos.
ChatGPT, Claude y Gemini como generadores de código
Comparamos los tres modelos aplicados a la misma tarea de Apps Script. Aprende cuál responde mejor a qué tipo de solicitud, cómo darle el contexto correcto y cómo pedirle que corrija su propio código cuando falla.
Ciclo pedir → pegar → probar → corregir con IA
El flujo profesional real: describes lo que quieres en lenguaje natural, pegas el código generado, lo ejecutas, copias el error que aparece de vuelta a la IA y le pides que lo corrija. Sin foros, sin tutoriales, sin frustraciones.

Cómo Pedirle Scripts a la IA: Prompts que Funcionan

Dar contexto de la estructura de tu hoja
La IA genera código mucho mejor cuando le describes tu hoja: nombres de columnas, en qué hoja están los datos, qué quieres que haga y cuál es el resultado esperado. Aprende la plantilla de prompt que usamos en cada ejercicio del curso.
Pedir scripts incrementales, no todo de golpe
Estrategia clave: divide el problema en pasos pequeños. Primero pídele que lea los datos, luego que los filtre, luego que envíe el correo. Scripts más cortos y simples son más fáciles de verificar y corregir.
Pedirle a la IA que explique su propio código
Pídele siempre que incluya comentarios en español línea por línea. Así entiendes qué hace cada parte sin necesitar conocimientos previos de JavaScript y puedes modificar los valores que te interesan.

Tipos de Scripts que Generamos en Esta Sesión

Scripts de lectura y escritura en Sheets
Le pedimos a la IA scripts que leen rangos de datos, los procesan con lógica condicional y escriben resultados en otras celdas u hojas. Base de casi cualquier automatización.
Triggers generados con IA
Describimos el comportamiento que queremos (ejecutar cada lunes a las 8 AM, al editar una celda específica, al enviar un formulario) y la IA genera el trigger completo listo para activar.
Menús y botones personalizados
Generamos con IA scripts que añaden menús propios a la barra de Sheets y botones que cualquier miembro del equipo puede usar sin tocar el editor de código.
Scripts de formato y limpieza de datos
Pedimos a la IA scripts que normalizan texto, eliminan duplicados, aplican formatos condicionales por código y reorganizan datos automáticamente al llegar nueva información.

Depurar y Mejorar Scripts con la Misma IA

El error como prompt: copiar y pegar fallas
Cuando un script falla, el registro de ejecuciones muestra el mensaje de error exacto. Aprende a copiarlo junto con el código a la IA y a pedirle que identifique y corrija el problema — sin entender el error tú mismo.
Pedir mejoras y variaciones del mismo script
Una vez que un script funciona, le pides a la IA que lo mejore: que agregue manejo de errores, que funcione para múltiples hojas o que sea más rápido. Iteras sobre código que ya sabes que sirve.

Proyectos reales: tableros, correos y más
Concepto Clave
Todo el código de esta sesión lo genera la IA. El objetivo no es que aprendas JavaScript — es que salgas con sistemas funcionando: un tablero conectado a tu base de datos en Sheets, un sistema de correos masivos y otras automatizaciones reales construidas en vivo durante la clase.

Proyecto 1 · Tablero de Control enlazado a Base de Datos en Sheets

Diseño de la arquitectura con la IA
Le describimos a ChatGPT o Claude la estructura del tablero que queremos: qué métricas mostrar, desde qué hojas tomarlas y cómo presentarlas. La IA propone la arquitectura de hojas y el script que las conecta.
Script que actualiza el tablero automáticamente
Generamos con IA un script que lee la hoja de base de datos, calcula totales, promedios y comparativas, y escribe los resultados en la hoja del tablero. Un trigger por tiempo lo ejecuta cada vez que se actualiza el archivo.
Formato condicional por código
La IA genera el script que colorea celdas del tablero automáticamente: verde si la meta se cumple, rojo si hay alerta, amarillo si está en riesgo. El tablero se autoactualiza visualmente sin intervención.
Botón "Actualizar tablero" generado con IA
Agregamos un botón visible en la hoja del tablero que cualquier usuario puede presionar para forzar la actualización manual. La IA genera el script del botón y el de la actualización en una sola solicitud.

Proyecto 2 · Correos Masivos desde una Base de Datos en Sheets

Base de datos de contactos en Sheets
Estructura correcta de la hoja de contactos: nombre, correo, empresa, estado del envío y columna de personalización. La IA nos ayuda a diseñar la hoja para que el script la lea sin errores.
Script de envío masivo con personalización por fila
Generamos con IA un script que recorre cada fila de la base de datos, arma un correo personalizado con el nombre y datos del destinatario y lo envía por GmailApp. Cada correo es diferente aunque el proceso sea automático.
Control de envíos: marcar "enviado" por fila
El script escribe "Enviado ✓" en la columna de estado al terminar cada correo. Si el proceso se interrumpe, al reanudarlo solo procesa las filas sin marcar — nunca se manda dos veces al mismo contacto.
Envío condicional por filtro
Le pedimos a la IA que el script solo envíe correos a filas que cumplan una condición: clientes activos, prospectos sin respuesta o registros de una fecha específica. Campañas segmentadas sin herramientas externas.
// Prompt que usamos en clase para generar este script: // "Crea un script de Google Apps Script que lea la hoja 'Contactos'. // Las columnas son: A=Nombre, B=Email, C=Empresa, D=Estado. // Para cada fila donde D esté vacío, envía un correo a B con asunto // 'Hola [Nombre]' personalizado y escribe 'Enviado' en la columna D."

Proyecto 3 · Más Automatizaciones Generadas en Vivo con IA

Notificación automática al responsable por alerta
Script generado con IA que revisa una columna de indicadores y envía un correo de alerta al responsable cuando algún valor supera o baja del umbral definido. Se ejecuta automáticamente cada día con un trigger.
Reporte semanal de KPIs por correo
La IA genera un script que todos los lunes a las 8 AM lee los datos del tablero, construye un resumen en HTML con los KPIs de la semana y lo envía automáticamente a los correos del equipo directivo.
Registro automático de cambios en la hoja
Script con onEdit que registra en una hoja de auditoría quién editó qué celda, qué valor tenía antes y cuál tiene ahora. Trazabilidad completa de cambios sin ningún proceso manual. La IA lo genera en menos de un minuto.
Importar datos externos y consolidar en Sheets
Generamos con IA un script que descarga datos desde una URL pública (CSV o JSON), los parsea y los pega en una hoja de Sheets automáticamente. Actualización de datos externos sin copiar y pegar nunca más.
