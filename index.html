Tu Ruta de Aprendizaje (3 Sesiones Prácticas)

📋 Sesión 1: Fundamentos

  • Manejo de archivos (Importar, Compartir, Publicar)
  • Inmovilizar filas, formato y diseño profesional
  • Protección de datos y validaciones
  • Rangos con nombre y referencias (Relativas/Absolutas)
  • Filtros, vistas y tablas dinámicas

🔧 Sesión 2: Fórmulas Poderosas

  • Funciones esenciales: SUMA, PROMEDIO, MAX, MIN
  • Manipulación de texto: SPLIT, JOIN, MAYUSCULAS
  • UNIQUE, BUSCARV, BUSCARH, TRANSPONER
  • Condicionales avanzados: SI, SI.ERROR, SUMAR.SI
  • ARRAYFORMULA y manejo de arreglos

🚀 Sesión 3: Nivel Experto

  • FILTER, SPARKLINE para visualización rápida
  • IMPORTRANGE para conectar hojas
  • QUERY completo (Select, Where, Order By, Group By, Pivot)
  • Macros para automatización
  • Hipervínculos y ARRAY_CONSTRAIN
